    Bizyness is an accounting software that helps entrepreneurs to manage their activities with ease.

    Introduction

    This API provides endpoints to manage the data in your Bizyness account.

    Authentication

    Bizyness offers two forms of authentication: - API Key - JWT Token

    Base URLs:

    • https://virtserver.swaggerhub.com/lpb/bizyness-api/1.0.0

    • https://api.bizyness.fr

    Authentication

    • API Key (ApiKey)

      • Parameter Name: X-Api-Key, in: header.

    • HTTP Authentication, scheme: bearer
